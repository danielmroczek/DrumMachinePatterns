# [260 Drum Machine Patterns - Rene-Pierre Bardet](https://www.amazon.com/Drum-Machine-Patterns-Leonard-Corp/dp/0881888877)

## Sound Legend
```
BD: Bass Drum
SD: Snare Drum
LT: Low Tom
MT: Medium Tom
HT: High Tom
CH: Closed Hi-Hat
OH: Open Hi-Hat
CY: Cymbal
RS: Rim Shot
CP: Claps        // CPS in book
CB: Cowbell
TM: Tambourine   // TAM in book
```

## Raw Format

```
$name $signature
--------------------
<guide>
$accent
$sound_1: $pattern_1
$sound_2: $pattern_2
...
$sound_n: $pattern_n
```

